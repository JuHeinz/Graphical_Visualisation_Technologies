Einsendeaufgabe 5

Kamerabewegung

Kamera-Bewegung:
  • Pfeiltase → : Kamera gegen Uhrzeigersinn drehen
  • Pfeiltase ← : Kamera im Uhrzeigersinn drehen
  • n : Radius vergrößern
  • Shift + n : Radius verkleinern
Kamera-Projektionen:
  • p : Perspective
  • o : Orthogonal
  • f : Frustum

Der Code für die Rekursion ist eine Übersetzung des verlinkten Materials von Andreas Kahler von C# auf Javascript.
